1. **Clone the repository**

   ```bash
   git clone https://github.com/igenius99-dev/ISA-at-ASU.git
   cd ISA-at-ASU
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
